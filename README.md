# NeuroMarket-EEG-Based-Purchase-Prediction-System
NeuroMarket is an Explainable AI (XAI) framework that predicts consumer purchase decisions using Electroencephalography (EEG) brain signals from the SEED dataset. The system extracts frequency-band features from EEG recordings and trains an  XGBoost classifier using Leave-One-Subject-Out  LOSO  Cross-Validation to evaluate performance..
